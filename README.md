## Mermaid Architecture Library

### GitOps Delivery Flow

```mermaid
flowchart LR
    A["Developer Commit"] --> B["GitHub Actions / Jenkins"]
    B --> C["Build and Test"]
    C --> D["Container Image"]
    D --> E["Security Scan and CVE Checks"]
    E --> F["Helm Chart Update"]
    F --> G["ArgoCD Sync"]
    G --> H["Kubernetes Rollout"]
    H --> I["Prometheus and Grafana Validation"]
    I --> J["SLO / SLI Review"]
```

### AI Agent Workflow

```mermaid
flowchart TD
    U["User Query"] --> API["Agent API"]
    API --> INTENT["Intent Detection"]
    INTENT --> POD["Pod Diagnostics"]
    INTENT --> NODE["Node Diagnostics"]
    INTENT --> SVC["Service Diagnostics"]
    POD --> RAG["RAG Context"]
    NODE --> RAG
    SVC --> RAG
    RAG --> CACHE{"Redis Cache Hit?"}
    CACHE -- "Yes" --> ANSWER["Response"]
    CACHE -- "No" --> LLM["Gemini / OpenAI API"]
    LLM --> ANSWER
    ANSWER --> OBS["Trace, Logs, Metrics"]
```

### Observability Stack

```mermaid
flowchart TB
    APP["Applications"] --> OTEL["OpenTelemetry"]
    APP --> PROM["Prometheus"]
    APP --> LOKI["Loki"]
    OTEL --> TRACE["Distributed Traces"]
    PROM --> GRAF["Grafana Dashboards"]
    LOKI --> GRAF
    TRACE --> GRAF
    PROM --> ALERT["Alertmanager"]
    ALERT --> INCIDENT["Incident Response"]
    INCIDENT --> RCA["RCA and Remediation"]
```
